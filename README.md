# Volley

Volley is a polished Pong-style arcade game built with Python and Pygame CE, with menus, AI difficulty tiers, rally survival, persistent settings, generated sound, visual feedback, and a PyInstaller build path.

## Features

- Main menu, mode select, difficulty select, countdown, pause, settings, controls, and game-over screens.
- 1-player vs AI, local 2-player, and rally survival modes.
- Rookie, Pro, and Legendary AI tiers with different reaction and prediction behavior.
- Sub-stepped swept collision, paddle clamps, speed cap, edge-influenced bounce angles, win-by-two scoring.
- Persistent settings and rally high scores through `platformdirs`.
- Generated original WAV sound effects and a simple looping music track.
- Fixed-resolution internal renderer with window scaling and fullscreen toggle.

## Run from source

```bash
python -m pip install -e ".[dev]"
python scripts/generate_sfx.py
python -m volley.main
```

## Tests

```bash
SDL_VIDEODRIVER=dummy SDL_AUDIODRIVER=dummy python -m pytest
```

On Windows PowerShell:

```powershell
$env:SDL_VIDEODRIVER="dummy"; $env:SDL_AUDIODRIVER="dummy"; python -m pytest
```

## Build

```bash
python -m PyInstaller --noconfirm --name Volley --windowed --collect-data volley src/volley/main.py
```

The executable lands in `dist/Volley/` on the current OS.

## Credits

All audio assets are procedurally generated by `scripts/generate_sfx.py`. The game uses system fonts by default and does not bundle third-party media. `pygame-ce` is used because it is actively maintained and keeps the standard `pygame` import API.

## Known Gaps

Online multiplayer, gamepad support, replay/spectate mode, tournament brackets, and touch controls are intentionally tracked as future work.
